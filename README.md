# SNMI Network Visualization

Interactive network visualization of SNMI stakeholder relationships using SigmaJS.

## Features

- **Interactive Graph**: Pan, zoom, and click nodes to explore relationships
- **Search**: Find people and organizations by name
- **Filters**: Filter by node type and status
- **Node Details**: Click any node to see detailed information
- **Statistics**: Real-time stats showing network metrics

## Data

- **239 nodes**: People, organizations, healthcare entities, and more
- **408 edges**: Relationships and connections between entities

## Usage

Simply open `index.html` in a web browser to view the visualization.

For GitHub Pages, the visualization will be available at: `https://[your-username].github.io/[repo-name]/`

## Technology

- [SigmaJS](https://www.sigmajs.org/) - Network visualization
- [Graphology](https://graphology.github.io/) - Graph data structure
- ForceAtlas2 - Force-directed layout algorithm
